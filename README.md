📌 Project Showcase Web Application

📖 Description

This project is a web-based application that allows users to showcase their projects by adding details such as title, domain, description, and GitHub repository link.

The application helps students explore different types of projects across various domains like Computer Technology, Artificial Intelligence, Data Science, Mechanical, Civil, and Electrical.

It serves as a platform to understand project ideas and get guidance on how to start new projects.

---

🚀 Features

- 🏠 Home page with domain selection
- 📂 View projects based on selected domain
- ➕ Add new project with details
- 📄 View all projects
- 🔗 Direct link to GitHub repositories
- 💾 Data stored in MongoDB database
- 🎨 Simple and clean user interface

---

🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS (Embedded JavaScript)
- HTML & CSS

⚙️ Installation & Setup

1. Clone the repository

git clone <your-repo-link>

2. Navigate to project folder

cd project-showcase

3. Install dependencies

npm install

4. Start MongoDB locally

5. Run the application

node app.js

6. Open browser

http://localhost:3000
